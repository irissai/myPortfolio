Web Developer

กัญฐิมา เจริญ

ดิฉันเป็นนักศึกษาจบใหม่จากสาขาเทคโนโลยีสารสนเทศ วิทยาลัยการคอมพิวเตอร์ มหาวิทยาลัยขอนแก่น มีความสนใจในด้านการพัฒนาเว็บไซต์ โดยเฉพาะการเขียนโปรแกรมด้วย HTML, CSS, JavaScript, React และ Node.js

ดิฉันชื่นชอบการเรียนรู้เทคโนโลยีใหม่ ๆ อยู่เสมอ และมีประสบการณ์ในการพัฒนาโปรเจกต์ทั้งจากรายวิชา และการแข่งขัน อาทิ การแข่งขัน National Software Contest (NSC) 2024

ขณะนี้กำลังมองหาโอกาสในการเริ่มต้นทำงานในตำแหน่ง Web Developer เพื่อนำความรู้ที่มีไปต่อยอด และเติบโตในสายงานเทคโนโลยีที่ตนเองชอบ

About Me


สวัสดีค่ะ ดิฉันชื่อ ทราย เป็นนักศึกษาจบใหม่จากสาขาเทคโนโลยีสารสนเทศ วิทยาลัยการคอมพิวเตอร์ มหาวิทยาลัยขอนแก่นดิฉันเริ่มสนใจด้านการพัฒนาเว็บไซต์ ตั้งแต่ช่วงปีที่ 3 ของการศึกษา ขณะทำโปรเจกต์จบ และได้ค้นพบว่าตนเองชอบในการออกแบบ และสร้างเว็บไซต์อย่างแท้จริง โดยเฉพาะการพัฒนาเว็บด้วย HTML, CSS, JavaScript, React และ Node.js ซึ่งเป็นเทคโนโลยีหลักที่ดิฉันใช้ในการเรียนรู้และพัฒนาโปรเจกต์ ทั้งในรายวิชาและกิจกรรมนอกห้องเรียน

นอกจากนี้ ดิฉันยังเป็นคนที่ชื่นชอบการเรียนรู้เทคโนโลยีใหม่ ๆ อยู่เสมอ และมักใช้เวลาว่างในการศึกษาเพิ่มเติมหรือทดลองพัฒนาโปรเจกต์เล็ก ๆ เพื่อฝึกฝนและต่อ ยอดความสามารถของตนเอง

เป้าหมายของดิฉันคือการเริ่มต้นอาชีพในสายงาน Web Developer และเติบโตเป็นนักพัฒนาที่สามารถสร้างเว็บไซต์ที่มีคุณภาพ ใช้งานง่าย และตอบโจทย์ความต้อง การของผู้ใช้งานได้อย่างแท้จริงค่ะ
