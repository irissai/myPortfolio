กัญฐิมา เจริญ

ดิฉันเป็นนักศึกษาจบใหม่จากสาขาเทคโนโลยีสารสนเทศ วิทยาลัยการคอมพิวเตอร์ มหาวิทยาลัยขอนแก่น มีความสนใจในสายงานด้าน Frontend และ Backend มีความสามารถในการเรียนรู้และปรับตัวกับเทคโนโลยีใหม่ ๆ ได้อย่างรวดเร็ว

ดิฉันชื่นชอบการเรียนรู้เทคโนโลยีใหม่ ๆ อยู่เสมอ และมีประสบการณ์ในการพัฒนาโปรเจกต์ ทั้งจากการทำโปรเจกต์จบและการแข่งขัน อาทิ การแข่งขัน National Software Contest (NSC) 2024

ขณะนี้กำลังมองหาโอกาสในการเริ่มต้นทำงานในตำแหน่ง Developer เพื่อนำความรู้และทักษะที่มีไปต่อยอด และเติบโตอย่างมั่นคงในสายงานด้านการพัฒนาเว็บไซต์ในอนาคต

ประวัติส่วนตัว

สวัสดีค่ะ ชื่อกัญฐิมา เจริญ ชื่อเล่น ทราย เป็นนักศึกษาจบใหม่จากสาขาเทคโนโลยีสารสนเทศ วิทยาลัยการคอมพิวเตอร์ มหาวิทยาลัยขอนแก่น ดิฉันเริ่มสนใจด้านการพัฒนาเว็บไซต์ ตั้งแต่ช่วงปีที่ 3 ของการศึกษา ขณะทำโปรเจกต์จบ และได้ค้นพบว่าตนเองชอบในการออกแบบและสร้างเว็บไซต์ โดยเฉพาะการพัฒนาเว็บด้วย HTML, CSS, JavaScript, React และ Node.js ซึ่งเป็นเทคโนโลยีหลักที่ดิฉันใช้ในการเรียนรู้และพัฒนาโปรเจกต์จบการศึกษา


นอกจากนี้ ดิฉันยังเป็นคนที่ชื่นชอบการเรียนรู้เทคโนโลยีใหม่ ๆ อยู่เสมอ และมักใช้เวลาว่างในการศึกษาเพิ่มเติมหรือลองพัฒนาโปรเจกต์เล็ก ๆ เพื่อฝึกฝนและต่อยอดความสามารถของตนเอง


เป้าหมายของดิฉันคือการเริ่มต้นอาชีพในสายงาน Developer และเติบโตเป็นนักพัฒนาที่สามารถสร้างเว็บไซต์ที่มีคุณภาพ ใช้งานง่าย และตอบโจทย์ความต้องการของผู้ใช้งานได้อย่างแท้จริงค่ะ
